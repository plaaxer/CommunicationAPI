Detailed Description

Defines universal port mappings for inter-component communication.

This class provides a centralized, static mapping of component/service types to predefined port numbers.

A component "subscribes" to its designated port to receive all messages directed to that service type.

Example Usage: ComponentPorts::Port gps_port = static_cast<ComponentPorts::Port>(ComponentPorts::ID::GPS_SENSOR);
