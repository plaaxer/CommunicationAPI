Detailed Description

template<typename UnitTag>
class DataGenerator< UnitTag >

Generates data for a Transducer. It is a simulation facility. This source can be replaced with a Network source later. For now, we will only use this to fetch random data. After the infrastructure consolidation, that class will be autonomous (running alone and notifying observers).
